# dstack prune cache

The `prune` command prunes the cache of a specific workflow.

## Usage

<div class="termy">

```shell
$ dstack prune cache --help

Usage: dstack prune cache [-h] WORKFLOW

Positional Arguments:
  WORKFLOW    A workflow name to prune cache
```

</div>
