Cost-effective ML development in your cloud

dstack is an open-source tool for managing dev environments and automating ML tasks on any cloud.

dstack supports all major cloud providers, including AWS, GCP, and Azure.

Running ML tasks in your cloud with ease

A task can be a web application, a training script, a data preparation job, etc.

Specify hardware resources (e.g., GPU and memory), and indicate if you want to use spot instances.

dstack automates the provisioning of cloud resources, runs the task, and cleans up the resources once it's finished or stopped.

If the task exposes any ports, dstack forwards them for secure and convenient access.

Run ML on any cloud easily and cost-effectively

Declarative configurations

Define dev environments and tasks declaratively as code. Run them on demand using the CLI, eliminating the need of using cloud UI and proprietary APIs, which can be a nightmare.

GPU cost optimization

Save costs by effectively utilizing cloud spot instances across multiple regions. Automatically recover from interruptions and track GPU usage, while enforcing limits per team.

Data-centric approach

Run dev environments and tasks where your data is located. Eliminate costs associated with ingress and egress data transfers. Ensure the security and privacy of your data and models.

No MLOps is required

dstack integrates with your cloud natively and doesn't require engineering. No hustle with Kubernetes or custom Docker images.

Cloud-agnostic

Use the same interface to work with any major cloud provider, including AWS, GCP, and Azure. Use multiple clouds to optimize costs.

Self-hosted

Run the dstack server on your local machine or easily deploy it, as a Docker image, anywhere within your environment, including your own cloud.

Integrates natively with all major clouds

Installation and setup take less than a minute

$ pip install "dstack[aws,gcp,azure]"
$ dstack start

And you're done! Now you and your team can run dev environments and tasks both locally and in your cloud.
